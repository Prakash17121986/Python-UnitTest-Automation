Steps:
1. Launch command prompt
2. Copy the scripts folder to Desktop
3. Go to Desktop folder.
4. Run the below command.
	python run.py

Note:
1. lib folder is package - it consist of module and other package
2. tests folder is package - it consist of list of individual modules
3. HTMLTestRunner folder is consist of templates folder, static folder with CSS style file and runner.py
4. run.py file have 2 class name with different class methods
	- create test suite
	- add test
	- call HTMLTestrunner method for runner
	- run the suite using runner
5. You can see HTML test report in report folder. You can view the Python Unittest Automation report for all the tests.
